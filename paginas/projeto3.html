A Proposta

A proposta desse trabalho era de criar um site contendo uma página inicial com menu, que redicionaria para outras três páginas:

1. Página com imagens.

2. Página com vídeos do youtube.

3. Página com componentes de input com senha, redirecionando para uma página livre.

O Site

Decidimos criar uma empresa que vende coxinhas chamada Coxinheiros. Criamos uma pequena história para colocar na página principal, em que a Coxinheiros é uma empresa familiar, com receitas passadas por gerações desde 1975.

O site contém uma página de Cardápio com os mais variados sabores de coxinhas e respectivos preços para cada tamanho. Na aba "Nossas Receitas", inserimos dois vídeos do youtube de receitas. E por fim, na página "Faça seu Login", há apenas um formulário com a senha escrita logo abaixo, contendo os créditos finais do trabalho.
